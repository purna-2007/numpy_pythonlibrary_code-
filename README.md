# NumPy Python Library Codes

##  Project Purpose
This project contains beginner-friendly examples of the NumPy library.
It is designed to help understand numerical computing and array operations in Python.

##  Technologies Used
- Python
- Numpy
- Jupyter Notebook

##  Topics Covered
- Creating NumPy arrays
- Array indexing and slicing
- Mathematical operations
- Shape and reshaping of arrays
- Basic NumPy functions

##  How to Use
1. Download or clone this repository
2. Open 'numpy.ipynb' using Jupyter Notebook or Google Colab
3. Run the cells step by step to understand NumPy concepts
